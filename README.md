Created a Machine Learning-based House Price Prediction system using Python, Pandas, Scikit-learn, Matplotlib, and Seaborn to predict housing prices on a Kaggle dataset of 1,460 samples. Performed exploratory data analysis, feature engineering, and outlier removal to improve model performance, achieving an R² score of 0.87 using Random Forest Regression and reducing RMSE by 18% compared to baseline Linear Regression models.
